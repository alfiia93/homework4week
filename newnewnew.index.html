Пример карточки

Заголовок карточки

Это описание карточки. Здесь можно написать что-то интересное о товаре или услуге.

/* Общие стили для страницы */ body { font-family: Arial, sans-serif; display: flex; justify-content: center; align-items: center; height: 100vh; background-color: #f4f4f4; margin: 0; } /* Стили для карточки */ .card { width: 300px; border-radius: 10px; overflow: hidden; box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1); background-color: #fff; transition: transform 0.3s ease; } .card:hover { transform: scale(1.05); /* Эффект увеличения при наведении */ } /* Стили для изображения карточки */ .card-img { width: 100%; height: 200px; object-fit: cover; } /* Стили для контента карточки */ .card-content { padding: 20px; } .card-title { font-size: 1.5em; margin: 0; color: #333; } .card-description { font-size: 1em; color: #666; margin-top: 10px; }
